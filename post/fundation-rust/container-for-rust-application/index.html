Centos镜像作为运行时

镜像大小为245MB

使用这种方式打包好镜像后尝试运行,发现报错了,运行不了/lib64/libc.so.6: version 'GLIBC_2.18' not found

主要是由于Centos7中的glibc版本太低了,这里简单的做法是将centos7替换成了centos8解决了这个问题.
