介绍

本文会介绍在 CNCF 中的 Rust 项目,项目的介绍可能是很简陋的,如果你可以帮忙完善一下项目的介绍,那就太好了,随时欢迎在评论区或网站仓库提交 PR.

项目列表

项目按照项目名称首字母排序.

Akri

Akri最初由微软开源并贡献给 CNCF 的一个边缘计算项目,目前是 Sandbox 级别的项目.

Akri提供了一个类似于 CNI 的抽象层,但不是作用于网络而是用于查找,利用以及监控边缘设备,例如摄影机或传感器等小型设备.

foniod（epbf相关）

foniod 基于 eBPF 的一个安全监控代理.

krustlet

krustlet是一个 Rust 实现的 kubelet,可以在 kubernetes 中运行 WASM,即将发布 1.0 版本.

kube-rs

kube-rs是 Rust 实现的 kubernetes 客户端,同时还实现了一个用 kube-rs 编写的 controller 示例 controller-rs.

krator

krator是一个基于 kube-rs 实现的 operator 框架.目前还处于高度实验期,但是非常值得期待!

linkerd2-proxy

linkerd2-proxy 是一个 rust 实现的专用于 service mesh 的代理.

parsec

parsec

TiKV

TIKV

这个可能国内的相对比较熟悉一点,PingCAP 开源的一个 key value 存储.

tremor

tremor

wasmCloud

wasmCloud

curiefense

curiefense

Curiefense 是一款新型的应用安全平台，可以保护站点、服务和 api。它扩展了 Envoy 代理以防御各种威胁，包括 SQL 和命令注入、跨站脚本(XSS)、帐户接管(ato)、应用层 DDoS、远程文件包含(RFI)、API滥用等。

Xline

Xline 是一个兼容 Etcd API 的用于元数据管理的 KV 存储，目标是基于 CURP 协议实现跨云/跨数据中心管理元数据，由 datenlord 开源并捐赠到 CNCF,目前是 CNCF sandbox 项目.
