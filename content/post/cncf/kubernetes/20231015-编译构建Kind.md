# 前言  

本页面将专注于构建自定义的 Kind 容器镜像,只需要在下面评论区添加对应的命令评论触发构建即可.

也可以在 github discussion 地址来发评论,效果是一样的.

默认的命令评论如下:  

```shell
/build-kind
```

即可开始构建一个拉取当前 kubernetes/kubernetes 仓库的最新代码进行构建,构建完成后会将构建的容器镜像回复给对应的评论,尽情享用吧 :)
